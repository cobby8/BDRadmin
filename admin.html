대회 관리

종별 설정
수정 모드에서만 + 구분/종별 추가 버튼이 활성화됩니다. 칩을 클릭하면 제거됩니다.
대회장소
우측 + 장소 추가를 누르면 행안부 주소검색 팝업이 열립니다. 결과에서 선택하면 행이 추가됩니다.
대회기간
대회가 열리는 날짜 구간입니다.
접수기간
참가신청 접수 가능 시간입니다. (날짜+시간)
신규 생성 시 설정, 이후 수정/삭제 시 동일 코드 필요. (포커스 시에만 입력 가능)
상태미정
접수기간에 따라 자동 표시(접수전/접수중/마감). 관리자 수동 변경 불가.
